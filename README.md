# 🌐 Site Simples — HTML, CSS e JavaScript

Projeto desenvolvido durante o curso **"Crie um Site Simples usando HTML, CSS e JavaScript"** da [Fundação Bradesco].

---

## 📋 Sobre o projeto

Um site de lista de tarefas com alternância entre tema claro e escuro, desenvolvido do zero com HTML, CSS e JavaScript puro.

## ✨ Funcionalidades

- Lista de tarefas simples;
- Botão para alternar entre tema claro e escuro;
- Estilização com variáveis CSS;
- Interatividade com JavaScript.

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript

## 📁 Estrutura do projeto

```
site-simples/
│
├── index.html
├── main.css
└── app.js
```
